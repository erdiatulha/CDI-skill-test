# CDI-skill-test
##Soal 0

```
Buat fungsi sederhana untuk menghitung jumlah huruf hidup dan huruf mati dari suatu kalimat. Contoh:
input: "omama"   hasil: "huruf mati: 2, huruf hidup: 2"  (huruf hidup yaitu o dan a. a tidak perlu muncul 2x)
```

## Soal 1

```
Buat fungsi sederhana untuk mengurutkan abjad dari suatu kalimat. Contoh:
input: "omama"   hasil: "aammo" ('a' dan 'm' lebih dulu muncul dalam sistem abjad dari pada 'o')
input: "osama"   hasil: "aamos"
```

## Soal 2

```
Buat webservice untuk salah satu fungsi di atas (boleh pilih nomor 0 atau nomor 1)
```

Soal 3
```
Buat program klien untuk mengakses fungi di atas melalui webservice dari hasil nomor 2
```
